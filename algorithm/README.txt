README
========
TO RUN:
1. Please extract all the files. All java files should be placed in a folder called 'algorithm'.
2. Run the following command to compile all the java files

javac algorithm/*.java

3. Run the following command to run the program with the 3 given arguments

java algorithm/Main algorithm/human.txt algorithm/pig.txt algorithm/chicken.txt

or choose your own files:

java algorithm/Main path1 path2 path3
